# Smart Task Manager - Mobile App

## Overview
Android app built with Kotlin + Jetpack Compose.  
Features: offline caching (Room DB), task CRUD, modern UI, accessibility support.

## Tech Stack
- Kotlin
- Jetpack Compose
- Room Database
- Material Design

## Setup
1. Clone this repo
2. Open in Android Studio
3. Run on emulator or device

## Roadmap
- Backend API integration
